## A shared application layer

MapConductor separates your application’s geographic state from a provider’s rendering SDK. Your domain code creates `GeoPoint`, camera, and overlay states; a provider package supplies the map view and controller.

```tsx
const tokyo = GeoPoint.from({ latitude: 35.6812, longitude: 139.7671 });
const mapState = useMapLibreViewState({
  mapDesignType: MapLibreDesign.OsmBright,
  cameraPosition: MapCameraPosition.from({ position: tokyo, zoom: 12 }),
});

return (
  <MapLibreView state={mapState}>
    <Marker position={tokyo} />
  </MapLibreView>
);
```
