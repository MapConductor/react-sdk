## アプリケーション層を共有する

```tsx
const tokyo = GeoPoint.from({ latitude: 35.6812, longitude: 139.7671 });
const mapState = useMapLibreViewState({
  mapDesignType: MapLibreDesign.OsmBright,
  cameraPosition: MapCameraPosition.from({ position: tokyo, zoom: 12 }),
});

return (
  <MapLibreView state={mapState}>
    <Marker position={tokyo} />
  </MapLibreView>
);
```

[チュートリアル](/ja/getting-started/)から始め、[Web](/ja/platforms/web/) または [React Native](/ja/platforms/react-native/) の設定へ進んでください。
