# MapConductor React SDK documentation

This Astro + Starlight site is the documentation for the MapConductor React SDK. Its structure and visual foundation are based on the Android SDK documentation, while all content and examples target the TypeScript, React, and React Native packages in this repository.

## Development

```bash
cd docs
npm install
npm run dev
```

The development server is available at `http://localhost:4321`.

## Validation and production build

```bash
npm run check
npm run build
npm run preview
```

English pages live in `src/content/docs`. Japanese translations use the same path beneath `src/content/docs/ja`.
